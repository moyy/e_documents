# 功耗 && 带宽

## 1、功耗

### 1.1、移动端 比 PC 低 很多

+ 移动端，SoC：几瓦，一般 3瓦
+ PC端：100-200 瓦；

### 1.2、原因：手机设备体积小

+ 主因：电池续航
+ 次因：芯片面积
+ 次因：散热 --> 降频

### 1.3、内存读写：高功耗 操作

## 2、带宽（`BandWidth`）：瓶颈

+ 带宽：一秒钟能读多少字节内存？
	+ 带宽 = 频率 * 位宽 / 8 * 通道数 * 倍增系数
+ 例子：小米6, LPDDR 4x, 1866 MHZ, 32bit, 双通道
	- 带宽 = 1866 * (32 / 8) * 2 * 4 = 29.8 GB/s

手机 在 FLOPS 逼近主机，但 带宽 差距 还 很大。

||浮点 FLOPS|带宽|
|--|--|--|
|PS4|1.84T FLOPS|176 GB/s（位宽 256 bits）|
|Snapdragon 865|1.23T FLOPS|44 GB/s（同频率，位宽64bits）|

## 3、分辨率 与 带宽 关系

+ 分辨率 1920 * 1080
+ Color Buffer + DS Buffer共 2个 32位 缓冲
+ 每秒60帧，则 每秒 需要带宽 = 60 * 1920 * 1080 * 2 * (32/8) = 0.99 GB/s
